만약 패치가 제대로 진행되지 않는다면 "관리자 권한으로 실행"하여 재시도 해보십시오.

일부 백신이 확인되지 않은 프로그램이라며 패치를 차단합니다. 이 스크립트는 많은 스크립트 파일을 불러와 패치를 진행하기 때문에 이후에도 수차례 같은 이유로 스크립트가 차단당하게 되며, 1회 차단해제로는 완벽히 설치가 불가능한점을 알려드립니다.
이경우, 잠시 백신을 꺼두시는것이 정신건강에 이롭습니다.
